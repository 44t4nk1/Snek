#### Installing

```text
git clone https://github.com/44t4nk1/Snek
------------------ FOR ANDROID --------------------
flutter build apk
go to build/app/outputs/apk/release/app-release.apk
install the APK on your android device
-------------------- FOR iOS ----------------------
Connect your iPhone to your Compiler
Run and build the app
```
